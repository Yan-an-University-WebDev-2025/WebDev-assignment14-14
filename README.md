# 单词记忆卡片（Flashcard）应用

一款基于 Web 的高效单词记忆工具，采用**间隔重复原理**辅助科学记忆，支持卡片学习、管理、进度跟踪与数据统计，适配多设备使用场景。

## 项目介绍
在英语学习中，传统单词记忆方式效率低下、易遗忘，本项目借助**间隔重复原理**（基于艾宾浩斯遗忘曲线），设计并实现了一款可视化、交互友好的单词记忆卡片应用。用户可通过卡片翻转学习、标记掌握状态、筛选复习内容等功能，大幅提升单词记忆效率，同时实时跟踪学习进度。

## 功能特点
### 1. 卡片学习
- 首页默认展示单张卡片，正面显示单词，背面显示释义与例句
- 支持 **CSS 3D 翻转动画**：点击卡片即可实现流畅翻转，视觉效果自然
- 提供「认识」「不认识」按钮：点击后自动更新卡片状态并切换至下一张，逻辑符合间隔重复规则

### 2. 卡片管理
- 新增卡片：通过表单提交单词、释义、例句，表单验证确保字段完整性
- 数据存储：所有卡片数据持久化至 `localStorage`，页面刷新/关闭后数据不丢失
- 状态更新：支持在卡片列表直接切换「已掌握/未掌握」状态，同步更新复习时间
- 卡片删除：提供删除功能，含确认弹窗防止误操作

### 3. 学习进度与筛选
- 智能复习：标记「不认识」的卡片将缩短复习间隔（1 天后复习），优先重复出现
- 筛选功能：支持按「全部卡片」「未掌握」「已掌握」筛选，聚焦重点复习内容
- 状态可视化：卡片列表标注「已掌握/未掌握」标签，未掌握卡片默认排序前置

### 4. 数据统计
- 实时展示：总卡片数、已掌握卡片数、学习进度百分比
- 可视化进度：通过渐变进度条直观呈现掌握程度
- 操作反馈：添加、删除、状态切换等操作均有弹窗通知，提升交互体验

### 5. UI 与动画
- 响应式设计：适配 PC 端、平板等不同屏幕尺寸，自动调整布局
- 交互动画：卡片翻转、按钮 hover 效果、通知弹窗滑入滑出等，提升使用体验
- 简洁美观：采用清晰的视觉层级与色彩区分，操作门槛低

## 技术栈
- 前端核心：HTML5（语义化结构）、CSS3（3D 动画、响应式布局、过渡效果）、JavaScript（交互逻辑、数据处理）
- 数据存储：localStorage（本地数据持久化）
- 图标美化：Font Awesome（界面图标优化）
- 开发工具：VS Code
